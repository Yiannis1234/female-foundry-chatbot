Female Foundry Chatbot MVP

Prototype conversation experience showing how a Wix widget could talk to an LLM-backed service and log data.

    How it works

    1. User message is sent to /api/chat.
    2. Server finds relevant records from the Index dataset.
    3. The LLM generates a contextual answer (or uses a fallback).
    4. Conversation is logged for analytics and training.
    Demo only. Replace with Wix widget + secure backend before going live.
